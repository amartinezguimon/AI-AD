# Models Directory

This folder stores the AI weights used by the system. Some are included in the repo, others are auto-downloaded on first run.

## Included in Repo
| File | Size | Description |
|---|---|---|
| `engagement_model.pth` | 4 KB | Custom-trained PyTorch MLP (Engaged vs Away classifier). Generated by `src/training/train.py`. |

## Auto-Downloaded (not in repo)
These are downloaded automatically the first time you run `main.py`, `calibrate.py`, or `data_collector.py`. No manual action needed — just make sure you have internet on first run.

| File | Size | Source |
|---|---|---|
| `face_landmarker.task` | ~3.6 MB | [MediaPipe Face Landmarker](https://storage.googleapis.com/mediapipe-models/face_landmarker/face_landmarker/float16/latest/face_landmarker.task) |
| `pose_landmarker_lite.task` | ~5.5 MB | [MediaPipe Pose Landmarker](https://storage.googleapis.com/mediapipe-models/pose_landmarker/pose_landmarker_lite/float16/latest/pose_landmarker_lite.task) |

## Manual Download (if auto-download fails)
If you're behind a firewall or the auto-download fails, download the files manually from the links above and place them in this `models/` directory.

## YOLOv8
`yolov8n.pt` is auto-downloaded by the `ultralytics` library on first import. It is placed in the project root, not in this folder.
